How to start sending alerts to Grafana OnCall from Datadog

  1. 1. Navigate to the Integrations page from the sidebar
  2. 2. Search for webhook in the search bar
  3. 3. Enter a name for the integration, for example: grafana-oncall-alerts
  4. 4. Paste the webhook URL, then save 
    {{ alert_receive_channel.integration_url }}
  5. 5. Navigate to the Events page from the sidebar to send the test alert
  6. 6. Type @webhook-grafana-oncall-alerts test alert
  7. 7. Click the post button

Next steps:

  1. 1. Add the routes and escalations in Escalations settings
  2. 2. Check grouping, auto-resolving, and rendering templates in Alert Templates Settings
  3. 3. Make sure all the users set up their Personal Notifications Settings on the Users Page

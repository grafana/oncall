How to start sending alerts to Grafana OnCall from Pingdom

  1. 1. Go to https://my.pingdom.com/integrations/settings
  2. 2. Click "Add Integration".
  3. 3. Type: Webhook. Name: Amixr. URL:
    4. {{ alert_receive_channel.integration_url }}
  4. 4. Go to "Reports" -> "Uptime" -> "Edit Check".
  5. 5. Select Amixr integration in the bottom.
  6. 6. Click "Modify Check" to save.

Next steps:

  1. 1. Add the routes and escalations in Escalations settings
  2. 2. Check grouping, auto-resolving, and rendering templates in Alert Templates Settings
  3. 3. Make sure all the users set up their Personal Notifications Settings on the Users Page
